# ekiden数据站

ekiden 数据分析平台前端项目

## 快速开始

### 安装依赖
```bash
npm install
```

### 开发模式
```bash
npm run dev
```

### 远程开发模式
```bash
npm run remote
```

### 构建
```bash
npm run build
```

### 预览
```bash
npm run preview
```

## 技术栈

- React 19
- TypeScript
- Vite
- Tailwind CSS
- React Router
- Chart.js
- Lucide React

## 项目结构

```
src/
├── pages/       # 页面组件
├── components/  # 可复用组件
├── hooks/       # 自定义 hooks
├── utils/       # 工具函数
├── App.tsx      # 主应用组件
└── main.tsx     # 入口文件
```
